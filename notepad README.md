# ⚡ NewsPulse AI Agent

## Overview
NewsPulse AI Agent is a web-based intelligent news platform that:
- Fetches global news
- Calculates trust score
- Generates quizzes from news
- Maintains a leaderboard

## Tech Stack
- Python (FastAPI)
- React (Vite)
- JavaScript
- Google News RSS

## Run Backend
```bash
cd backend
python -m uvicorn main:app --reload --port 8000
```

## Run Frontend
```bash
cd frontend
npm run dev
```

Open:
http://localhost:5173
